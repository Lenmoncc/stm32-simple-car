# stm32-simple-car
基于STM32F103C8T6完成的智能小车  
代码实现：  
    1.蓝牙连接并控制  
    2.超声波测距避障  
    3.电机驱动  
    4.舵机驱动  
    5.四路循迹（数字量）  
    6.PID                                                                                                                                               
通过手机蓝牙app与蓝牙模块连接，发送对应的信息，实现是否开启循迹功能。
